# multi-bg-remove

Script to remove background from multiples images in once.

# Requirements
 - PyQt5
 - rembg
 - numpy
 - PIL

## Setup

Setup to run on Windows.

1. Clone this repo.
2. Run:
```
python -m venv .
./Scripts/activate
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
```
3. To run:
```
python main.py
```
